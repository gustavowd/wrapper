Exemplos de como utilizar bibliotecas em C em linguagem de programação interpretadas:

1. Python
  - Para compilar a biblioteca em C:
    ./compile.sh
  - Para executar o código em Python:
    python test.py
2. Java
  - Para compilar a biblioteca em C e o código em Java:
    ./compile.sh
  - Para executar o código em Java:
    ./execute.sh
3. C#
  - Para compilar a biblioteca em C e o código em C#:
    ./compile.sh
  - Para executar o código em C#:
    mono test.exe
